# KeenAI v0.1.0

First **0.1.x** prerelease — KB lifecycle, CSAT, import, and Dashboard stubs. API may still change until **v0.2.0**.

## Highlights

- **KB lifecycle**: conversation close → crystallize; `auto_index` FTS; golden eval (`pnpm kb:eval`) + nightly CI
- **CSAT**: conversation `rating` on agent PATCH + widget endpoint; crystallize only with explicit CSAT ≥ 4
- **Import**: `keenai import zendesk --kb` → `kb_documents`
- **Dashboard**: Custom Actions 4-step wizard stub; Help Center KB search page
- **Docs**: [DESIGN-CODE-AUDIT.md](../DESIGN-CODE-AUDIT.md), [DEPLOYMENT.md](../DEPLOYMENT.md), [GA.md](../GA.md) (0.1.0 checklist)

## Upgrade from v0.0.1 (Alpha)

```bash
git pull
pnpm install
pnpm db:migrate
```

Optional env for KB-19 LLM crystallize:

```bash
OPENAI_API_KEY=sk-...
KEENAI_CRYSTALLIZE_MODEL=gpt-4o-mini
```
