# KeenAI v0.0.1 (Alpha)

首个公开 Alpha 版本 — 自托管客服 MVP。

## 亮点

- **Inbox**：会话列表、详情、回复、内部备注、SSE / WebSocket
- **Messenger Widget**：Preact 嵌入组件、HMAC 身份验证
- **Email Channel**：IMAP 拉取、SMTP 发送、Webhook 入站
- **Copilot**：多 LLM Provider 草稿（OpenAI / Anthropic / DeepSeek / Kimi / Gemini / Ollama）
- **Workflows**：`first_message` / `customer_unresponsive` 触发、React Flow Builder
- **Docker Lite**：`docker compose --profile lite up --build` 一键部署

## 快速开始

```bash
cp .env.example .env
pnpm install && pnpm db:migrate && pnpm seed
pnpm dev    # API :8090 · Dashboard :3000
```

演示账号：`owner@keenai.local` / `keenai-demo-12`

## 已知限制

- 仅 LibSQL / SQLite 方言
- Workflow MVP（3 Trigger · 3 Block）
- 无 Tickets / Feedback Portal / Help Center

详见 [ALPHA.md](../ALPHA.md)。
